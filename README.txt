# EEBotBuilder
EEBotBuilder is a bot constructing tool for Everybody Edits designed by realmaster42. It uses it's own programming language and is used to make bots easily.

If you wish to find more information about it, go to https://forums.everybodyedits.com/viewtopic.php?pid=661714#p661714
Build the project in your own computer's source editor. It's safer, nicer and easier to work with.
To do this, clone it or download the source folder as zip/rar/7zip... and then unpack the zip to run the bot source so you can modify it.
